# Duck Hunt Game

A classic Duck Hunt game built with HTML5 Canvas.

[Play the game](https://srjuan23.github.io/Duck-Hunt-Game/)
## How to Play

### On Desktop
- Click "JUGAR" to start
- Move mouse to aim
- Click to shoot

### On Mobile
- Tap "JUGAR" to start
- Touch and drag to aim
- Tap to shoot

## Game Modes
- **Infinite Mode**: Ducks spawn infinitely
- **Game Over**: When 5 ducks escape
